# Deus Brand Colors

## Primary Palette

| Name | Hex | Usage |
|------|-----|-------|
| Ember | `#E8723A` | Primary orange — warm, inviting |
| Flame | `#F4A261` | Secondary orange — highlights, gradients |
| Deep Teal | `#1B7A6E` | Primary teal — contrast, depth |
| Ocean | `#2EC4B6` | Secondary teal — accents, gradients |

## Neutrals

| Name | Hex | Usage |
|------|-----|-------|
| Night | `#1A1A2E` | Dark backgrounds |
| Charcoal | `#16213E` | Dark mode surfaces |
| Slate | `#4A4E69` | Muted text, borders |
| Cloud | `#F8F9FA` | Light backgrounds |
| White | `#FFFFFF` | Light surfaces |

## Gradients

- **Möbius Flow**: `#E8723A` → `#F4A261` → `#2EC4B6` → `#1B7A6E` (the main logo gradient, follows the loop direction)
- **Warm Glow**: `#E8723A` → `#F4A261` (buttons, CTAs)
- **Cool Depth**: `#1B7A6E` → `#2EC4B6` (secondary elements)
