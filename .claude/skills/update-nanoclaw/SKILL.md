# Update Deus (Deprecated)

> **This skill is deprecated.** Deus no longer uses an upstream NanoClaw repository for updates. All channel code has been rewritten as MCP packages and lives in this repo.
>
> To update Deus, pull from your own origin remote:
>
> ```bash
> git pull origin main
> npm install
> npm run build
> ```
